Frontend - Uppgift 1

Uppgiften

Uppgiften innehåller några kriterier:

  • Det ska vara en webbplats som innehåller HTML, CSS, JavaScript och bilder.
  • Du skall använda NodeJs för att automatisera uppgifterna.
  • Installerade paket ska finnas med i en package.json-fil.
  • Det ska gå att återställa din applikation med npm install.
  • De uppgifter som skall automatiseras är:
    • Minifiering av CSS.
    • Transpilering och minifiering av JavaScript.
    • Minifiering av HTML.
    • Optimering av bilder - skall ske automatiskt vid uppdatering av bilder.
    • Bakåtkompabilitet i genererade filer, så att exempelvis JavaScript transpileras så att dessa fungerar i så många webbläsare som möjligt.
    • Kopiering av arbetsfiler till mapp för publicering med optimerade filer
  • Webbplatsen skall bestå av minst tre undersidor
